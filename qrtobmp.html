// Freeware — QR Code Generator

QRtoBMP無料

テキストを QRコードの BMP画像に変換する、コマンドラインツール

テキストファイルを読み込み、その内容を QRコード化してモノクロ BMP画像として出力します。 文字コードは UTF-8 / Shift-JIS / UTF-16 を自動判別。インストール不要・追加ランタイム不要。 業務システムやバッチ処理（.bat）に組み込んで、QRコードを自動生成する用途に最適です。

Windows XP〜11 32bit EXE インストール不要 文字コード自動判別 Shift-JIS 対応 MIT License

// Features

QRtoBMP の特長

業務の現場で「そのまま使える」ことを重視したシンプルなツールです

単一EXEで動作

インストーラ不要。EXEをコピーするだけで動きます。レジストリも一切使用しません。

追加DLL不要

VC++ランタイム等の再頒布パッケージは不要。OS標準のDLL（kernel32）のみで動作します。

文字コード自動判別

UTF-8 / Shift-JIS / UTF-16 を自動判別。既存システムが出力したSJISテキストもそのまま渡せます。

バッチ処理に最適

用途別の終了コードを返すため、.bat の ERRORLEVEL で結果を判定し自動処理に組み込めます。

Windows XP〜11対応

Windows XP SP3 から Windows 11 まで、32bit / 64bit のいずれでも動作します。

オープンソース（MIT）

MITライセンスのフリーソフト。商用・業務利用も無償でご利用いただけます。

// How it works

テキスト1つで、QR画像が完成

入力テキストを渡すだけ。スマホのカメラでそのまま読み取れる QRコードを生成します

▲ 実際の出力例（読み取り先: https://rapid-apps.co.jp/qrtobmp.html）

1

テキストファイルを用意

QRコードにしたい文字列を UTF-8 のテキストファイルに保存します。

2

コマンドを1行実行

QRtoBMP.exe input.txt qrcode.bmp を実行するだけ。

3

BMP画像が出力される

1モジュール8ピクセル・余白付きの読み取りやすいモノクロBMPが生成されます。

// Usage

使い方

コマンドプロンプトから次の書式で実行します

コマンドプロンプト 
REM 書式
QRtoBMP.exe <入力テキスト> <出力BMP>

REM 例：input.txt の内容を qrcode.bmp に出力
QRtoBMP.exe input.txt qrcode.bmp

文字コードは自動判別されます

入力ファイルの文字コードを自動で見分けるため、オプション指定は不要です。対応する文字コードは次の通りです。

文字コード備考
UTF-8BOMあり／なし どちらも可
Shift-JISCP932。日本語Windows標準の文字コード
UTF-16BOMあり。LE／BE どちらも可

判別は「BOM → ASCII → UTF-8 → Shift-JIS」の順に行い、判別後は内部でUTF-8に統一してQRコードを生成します。 いずれにも該当しない場合はエラー（終了コード5）となります。

バッチ処理での結果判定（終了コード）

処理結果は終了コード（ERRORLEVEL）で返るため、バッチファイルに組み込んで自動判定できます。

コード意味
0正常終了
1引数エラー（指定が不足／過多）
2入力ファイルの読み込み失敗
3QRエンコード失敗（データ長が上限を超過 など）
4出力ファイルの書き込み失敗
5文字コードの判別失敗（対応外の文字コード・バイナリファイル等）
run.bat 
QRtoBMP.exe input.txt out.bmp
if errorlevel 1 echo エラーが発生しました & goto :eof
echo QRコードを生成しました
入力がすべて半角英数字記号（ASCII）の場合は互換性を優先して ECIなしで生成し、 日本語など非ASCII文字を含む場合は ECI 26（UTF-8）付きで生成します。 Shift-JISで入力した場合も内部でUTF-8に変換してから生成するため、 いずれもスマートフォンのカメラ／QRリーダーアプリで読み取れます。
